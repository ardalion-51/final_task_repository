Git

  • git init- Создать пустой репозиторий Git или вновь инициализировать существующий
  • git status - Просмотреть статус нужного репозитория можно по ключевому слову status: его действие распространяется на подготовленные, неподготовленные и неотслеживаемые файлы.
  • git diff- Перечисляет все новые или изменённые файлы, которые нуждаются в фиксации
  • git add [файл] - Показывает различия по внесённым изменениям в ещё не проиндексированных файлах
  • git reset [файл]- Отменяет индексацию указанного файла, при этом сохраняет его содержимое
